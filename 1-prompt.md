Você deve criar uma aplicação frontend de calculadora genérica funcional em React, TypeScript e Vite.

Leia o arquivo BENCHMARK_CONTEXT.md antes de começar.

Requisitos funcionais:

1. Criar um projeto React com TypeScript e Vite.
2. Implementar uma calculadora com:
   - Soma
   - Subtração
   - Multiplicação
   - Divisão
   - Porcentagem
   - Decimal
   - Alternar sinal positivo/negativo
   - Limpar tudo
   - Apagar último caractere
   - Resultado
3. A calculadora deve suportar entrada por clique e teclado.
4. A calculadora deve tratar divisão por zero exibindo erro amigável.
5. A interface deve ser responsiva e utilizável em desktop e mobile.
6. A interface deve ter acessibilidade básica:
   - Botões com nomes claros
   - Estados visuais de foco
   - Estrutura semântica simples
7. Criar testes automatizados cobrindo:
   - Operações básicas
   - Divisão por zero
   - Entrada por teclado
   - Limpar cálculo
8. Adicionar scripts no package.json para:
   - dev
   - build
   - test
   - lint
9. Atualizar README com:
   - Descrição do projeto
   - Como instalar
   - Como rodar
   - Como testar
   - Decisões técnicas principais

Critérios importantes:

- Não use bibliotecas de UI prontas.
- Não implemente funcionalidades fora do escopo.
- Não deixe o projeto quebrado.
- Ao terminar, rode build e testes.
- Se algum comando falhar, corrija antes de finalizar.
- Ao final, entregue um resumo objetivo do que foi feito, arquivos principais alterados e comandos de validação executados.
